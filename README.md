# ⛩ 옥문강 (Prison Realm)

> 화면을 봉인하는 스크린 타임 디톡스 앱

---

## 이 앱은 안전한가요?

**네, 안전합니다.**

이 앱의 **소스 코드 전체가 이 GitHub 저장소에 공개**되어 있습니다.
코드를 직접 읽고 어떤 동작을 하는지 확인하실 수 있습니다.

Windows에서 설치 시 "알 수 없는 게시자" 경고가 뜰 수 있습니다.
이는 바이러스가 있어서가 아니라, 개인 개발자가 만든 앱에는 값 비싼 공식 인증서를 붙이기 어렵기 때문입니다.
오픈소스 개인 프로젝트에서 흔히 발생하는 경고입니다.

**이 앱이 하는 일:**
- 설정한 시간(기본 50분) 동안 타이머를 실행합니다.
- 시간이 끝나면 화면 전체를 영상으로 덮는 오버레이를 표시합니다.
- 설정한 휴식 시간(기본 10분)이 끝나면 오버레이가 해제됩니다.
- 트레이 아이콘으로 상주하며, 인터넷 연결이 필요 없습니다.

**이 앱이 하지 않는 일:**
- 개인정보 수집 없음
- 외부 서버 통신 없음
- 파일 접근은 사용자가 직접 선택한 영상 파일만 해당

## 다운로드

[Releases](../../releases) 페이지에서 최신 버전을 받으세요.

설치 방법:
1. `옥문강 Setup x.x.x.exe` 다운로드
2. 실행 → "추가 정보" → "실행" 클릭 (SmartScreen 경고 통과)
3. 설치 완료 후 트레이 아이콘에서 시작

---

## 기능

- **타이머** — 활동 시간 / 봉인 시간 자유롭게 설정
- **프리셋** — 뽀모도로(25/5), 기본(50/10) 등 저장 가능
- **전체화면 오버레이** — 멀티 모니터 전부 봉인
- **커스텀 영상** — MP4 · WebM · GIF · MOV 사용 가능
- **다국어** — 한국어 · English · 日本語 · 中文
- **문구 커스터마이즈** — 경고 문구, 해제 버튼 텍스트 등 변경 가능

## 직접 빌드하기

```bash
git clone https://github.com/PyeongBee/digital_detox_prison_realm.git
cd digital_detox_prison_realm
npm install
npm run build
```

빌드 결과물은 `dist/` 폴더에 생성됩니다.
